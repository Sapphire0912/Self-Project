Laboratory Project

心臟超音波影像辨識
(2021.05 ~ 2022.12)

專案目標
  • 建構醫療系統輔助醫生診斷特定疾病
  • 根據不同 view ，分析該 view 在醫學上可診斷的資訊
  • 優先計算 Apical four chamber view 的 LVEF
系統架構
使用環境和工具
  • Pycharm Community Edition 2021
  • 利用 C 優化演算法效率
  • 利用 Python、OpenCV 開發影像辨識演算法
  • 利用 Scikit-Learn 開發機器學習模型
專案進度統整

★ 內容中的 A4C 即為 Apical Four Chamber

  • 2021.05 ~ 2021.08

    - 優化分類器程式的執行時間

    - 優化骨架化程式執行時間

    - 完成抓取影像 ROI 區域程式

  • 2021.09 ~ 2021.11

    - 完成 DCMToAVI.py 模組

    - 設計 Multi Threshold 演算法

    - 設計 A4C view 抓取二尖瓣位置的演算法

  • 2021.12 ~ 2022.01

    - 優化 Multi Threshold 演算法(部分以C語言實現)

    - 完成 MultiThreshold.py 模組

    - 修正、調整 A4C View 預測腔室及二尖瓣位置的演算法

    - 整合 ROI 區域程式、骨架化程式為 Preprocessing.py 模組

    - 完成系統架構 (v1.0)

  • 2022.02 ~ 2022.05

    - 利用 BFS 和 Connected component labeling 實現 A4C view 肌肉區域的演算法

    - 基於 A4C 的 K-means 結果、標準模型的關係，實現定義肌肉特徵點

    - 進行單元測試

    - 開發 Global Longitudinal Strain 演算法

  • 2022.06 ~ 2022.12

    - 統計 LVEF 醫生診斷資料

    - 新增專案管理中的測試數據及資料

    - 完成 API 文件 v1.0

    - 完成系統架構(v2.0) 的系統整合

    - 交接實驗室專案 (v2.0)
