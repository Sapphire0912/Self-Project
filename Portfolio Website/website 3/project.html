Laboratory Project

心臟超音波影像辨識
(2021.05 ~ 2022.12)

專案目標
  • 建構醫療系統輔助醫生診斷特定疾病
  • 根據不同 view ，分析該 view 在醫學上可診斷的資訊
  • 優先計算 Apical four chamber view 的 LVEF
系統架構
使用環境和工具
  • Pycharm Community Edition 2021
  • 利用 C 優化演算法效率
  • 利用 Python、OpenCV 開發影像辨識演算法
  • 利用 Scikit-Learn 開發機器學習模型
